# Zerlin Khadijah's Ear Piercing Ceremony Invitation

A beautiful animated digital invitation built with Next.js 14.

## Features
- Animated intro with floating earring & falling petals
- Event details screen
- RSVP flow with guest count
- Google Maps location reveal with confetti
- Sage green & gold theme with elegant typography

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Free Hosting on Vercel (Recommended)

**Vercel** is made by the creators of Next.js and offers completely free hosting.

### Steps:

1. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   # Create a new repo on github.com, then:
   git remote add origin https://github.com/YOUR_USERNAME/zerlin-invitation.git
   git push -u origin main
   ```

2. **Deploy on Vercel**
   - Go to [vercel.com](https://vercel.com) and sign up (free)
   - Click **"Add New Project"**
   - Import your GitHub repository
   - Click **"Deploy"** — that's it!

3. **Your site will be live at:**
   `https://zerlin-invitation.vercel.app` (or similar)

4. **Custom domain** (optional, also free on Vercel):
   - Go to Project Settings → Domains
   - Add your own domain like `zerlin2025.com`

## Other Free Hosting Options

### Netlify
1. Go to [netlify.com](https://netlify.com)
2. Drag & drop the project folder, or connect GitHub
3. It auto-detects Next.js and deploys

### Cloudflare Pages
1. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
2. Connect GitHub repo → Framework: Next.js → Deploy
